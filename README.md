# Rumble Royale — Telegram-бот

Бот для групповых чатов: участники жмут кнопку «Участвовать», по таймеру
запускается шуточная «битва на выживание» с раундами, и остаётся один
победитель. Итоги отправляются в чат игры и в личку администратору.

## Что умеет

- Управление из лички: `/start` → «Новая игра» → выбор чата → ввод минут сбора.
- Сообщение в группе с кнопкой «Участвовать» и живым списком игроков.
- Таймер сбора (любое число минут).
- Симуляция боя по раундам: убийства, двойные убийства, случайные смерти,
  редкие воскрешения — с эмодзи и разными фразами.
- Статистика игры (победитель + топ по убийствам) в чат и в личку админу.
- Статистика между играми НЕ сохраняется (по задумке).

## Установка

1. Создай бота у [@BotFather](https://t.me/BotFather), получи токен.
   - Убедись, что у бота включено добавление в группы (по умолчанию включено;
     при необходимости: BotFather → `/mybots` → бот → Bot Settings → Allow Groups).
2. Python 3.10+:
   ```bash
   pip install -r requirements.txt
   ```
3. Задай токен. Либо переменной окружения:
   ```bash
   export BOT_TOKEN="123456:ABC..."
   ```
   либо скопируй `.env.example` в `.env` и впиши туда токен.
4. Запусти:
   ```bash
   python bot.py
   ```

## Как пользоваться

1. Добавь бота в свою группу. **Рекомендуется сделать его администратором**,
   чтобы он гарантированно мог отправлять сообщения.
2. Открой личный чат с ботом, отправь `/start`.
3. Нажми «🎮 Новая игра» → выбери нужную группу (показываются только те, где
   ты администратор) → отправь число минут для сбора игроков.
4. В группе появится сообщение с кнопкой «Участвовать». Все, кто нажал,
   попадают в список.
5. Когда таймер истечёт, начнётся бой. Победитель и статистика придут в чат и
   тебе в личку.

Отменить незавершённую игру может админ группы кнопкой «Отмена» под сообщением.

## Настройки (необязательные env-переменные)

| Переменная   | По умолчанию | Описание                              |
|--------------|--------------|---------------------------------------|
| `BOT_TOKEN`  | —            | Токен бота (обязательно)              |
| `DB_PATH`    | `rumble.db`  | Файл со списком чатов бота            |
| `ROUND_DELAY`| `3`          | Пауза между раундами (сек)            |
| `MIN_PLAYERS`| `2`          | Минимум игроков для старта            |
| `MAX_MINUTES`| `1440`       | Максимальная длительность сбора (мин) |

## Структура проекта

- `bot.py` — хендлеры, сессии игр, сбор игроков, повествование.
- `game.py` — движок боя (чистая логика, без Telegram).
- `texts.py` — русские фразы и рендеринг событий/статистики.
- `db.py` — SQLite со списком чатов, где состоит бот.
- `config.py` — настройки из окружения.
- `test_game.py` — офлайн-проверка движка: `python test_game.py`.

## Заметки

- Бот хранит в `rumble.db` только список чатов (чтобы показывать их в выборе) —
  это не игровая статистика.
- Сообщения боя отправляются отдельными сообщениями с паузой `ROUND_DELAY` для
  драматизма; на больших играх это растягивает бой — уменьши паузу при желании.
- Имена игроков в фразах не склоняются (как и в оригинальном Rumble Royale),
  потому что это кликабельные упоминания.
